# TaskFlow v2.0

The ultimate "Flow State" desktop widget. Minimalist, keyboard-first, and distraction-free.

## Features

*   **Global Quick Capture**: Press `Alt+Space` anywhere to add tasks instantly.
*   **Zen Mode**: Focus on one task at a time (Eye icon).
*   **Smart Parsing**: Type *"Meeting tomorrow!"* to automatically set the date and priority.
*   **Notes**: Quick scratchpad for your thoughts.
*   **Always-on-Top**: Docks to your screen edge and auto-collapses when not in use.
*   **Keyboard First**:
    *   `Ctrl+N`: New Task
    *   `Ctrl+1/2`: Switch Tabs
    *   `Esc`: Collapse Window
