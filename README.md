# TaskFlow v5.3

The ultimate "Flow State" desktop widget. Minimalist, keyboard-first, and distraction-free.

## Features

*   **Global Quick Capture**: Press `Alt+Space` anywhere to add tasks instantly.
*   **Zen Mode**: Focus on one task at a time (Eye icon) with built-in Pomodoro timer.
*   **Smart Parsing**: Type *"Meeting tomorrow!"* to automatically set the date and priority.
*   **Dopamine Rewards**: Satisfying gold flash on completion and confetti bursts when finishing a section.
*   **Window Snapping**: Magnetically snaps to screen edges for perfect alignment.
*   **Notes**: Quick scratchpad for your thoughts.
*   **Always-on-Top**: Docks to your screen edge and auto-collapses when not in use.
*   **Automatic Updates**: Checks for new versions on startup and installs them.
*   **Settings**: Customize timer duration, auto-collapse, and window snapping.
*   **Smart Search**: Instantly filter tasks and notes with `Ctrl+F`.
*   **Keyboard First**:
    *   `Ctrl+N`: New Task
    *   `Ctrl+1/2`: Switch Tabs
    *   `Esc`: Collapse Window
